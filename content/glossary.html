막히는 단어 여기서. 전부 비유 한 줄.

웹·개발

용어= 사실은
프론트엔드사용자가 보는 화면
백엔드화면 뒤 데이터 처리 엔진
서버요청에 응답하는 24시간 켜진 컴퓨터
API프론트백엔드 데이터 주고받는 약속(메뉴판)
REST가장 널리 쓰는 API 형식(CRUD)
CRUD생성·조회·수정·삭제 4동작
JSON데이터를 주고받는 텍스트 형식
DB / 테이블데이터 창고(집) / 종류별 방
SQLDB에 명령하는 언어
IP / 도메인 / DNS숫자 주소 / 이름 주소 / 둘 변환기
배포(deploy)만든 걸 인터넷에 올려 남도 보게
클라우드서버를 빌려 쓰는 서비스
도커실행 환경 통째 포장(이사 박스)
프레임워크 vs 라이브러리골격 잡아주는 틀 vs 갖다 붙이는 도구
디버깅버그(문제) 원인 잡기 — 코딩 시간 80%
MVP최소 생존 기능(핵심 하나로 시작)
자료구조데이터를 효율적으로 저장하는 법(스택·큐·해시·연결리스트)

AI·에이전트

용어= 사실은
LLM거대 언어 모델(GPT·클로드 같은)
토큰LLM이 처리하는 텍스트 조각(비용 단위)
컨텍스트 윈도우한 번에 처리하는 정보량(작업 책상)
환각모르면서 아는 척 틀린 답
RAG답하기 전 자료 찾아보고 답하기(찾기+생성)
임베딩텍스트를 의미 담은 숫자로 변환
벡터DB임베딩(숫자) 저장·유사도 검색하는 DB
GraphRAG문서 관계를 지도(그래프)로 그려 검색
에이전트인식·판단·행동까지 하는 AI
오케스트레이션여러 에이전트의 흐름을 조율
서브에이전트일을 나눠 맡는 보조 AI
스킬AI에게 주는 업무 매뉴얼

하네스에이전트를 팀으로 엮는 틀
