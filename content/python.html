핵심 개념

1. 변수·자료구조

name = "철수"          # 타입 선언 불필요
nums = [1, 2, 3]       # list (순서)
user = {"name": "철수", "age": 20}   # dict (키-값)
tags = {"a", "b"}      # set (중복 없음)
user["name"]           # "철수"
쉽게: list=줄, dict=서랍(이름표), set=중복 없는 모음. JS의 배열·객체와 같은 개념.

2. 들여쓰기 = 블록

파이썬은 { } 대신 들여쓰기(4칸)로 코드 묶음 표시.

if age >= 20:
    print("성인")      # 들여쓰기 안쪽 = if에 속함
    print("입장 가능")
else:
    print("미성년")
쉽게: 줄 맞춤이 곧 문법. 들여쓰기 틀리면 에러 — 깔끔함이 강제됨.

3. 함수

def greet(name, times=1):     # times 기본값 1
    return f"안녕 {name}" * times   # f-string
greet("철수")                 # "안녕 철수"
쉽게: def로 함수 정의. f"{변수}"=문자열에 값 끼워넣기(편함).

4. 반복·컴프리헨션

for n in nums:
    print(n)
squared = [n*n for n in nums]   # 컴프리헨션 → [1,4,9]
evens = [n for n in nums if n%2==0]
쉽게: 컴프리헨션 = "한 줄 for". 리스트를 빠르게 변형·필터. 파이썬 시그니처.

5. 클래스 (OOP)

class Todo:
    def __init__(self, title):   # 생성자
        self.title = title
        self.done = False
    def toggle(self):
        self.done = not self.done

t = Todo("공부")
t.toggle()
쉽게: 붕어빵 틀(class)로 붕어빵(객체) 찍기. self=자기 자신. 데이터+기능 묶음.

필수 문법 총정리 — 한 파일, 줄별 주석

변수·자료형·함수·조건·반복·자료구조·클래스가 한 덩어리에. 주석 = 그 줄이 하는 일 + 어떻게 읽는지. 파이썬은 들여쓰기(스페이스 4칸)로 블록을 구분한다 — 중괄호 없음.

# ── 1. 출력과 주석 ──────────────────────────────
print("Hello")          # print(x) = x를 화면에 출력. #부터 줄 끝까지는 주석

# ── 2. 변수: 타입 선언 없이 바로 ────────────────
pi = 3.14               # 타입 안 씀 — 값 보고 알아서 판단(동적 타입)
count = 0               # 정수
count += 1              # count = count + 1 축약 → 1

# ── 3. 자료형 ───────────────────────────────────
num = 42                # int(정수)
flt = 3.5               # float(소수)
txt = "글자"             # str(문자열)
ok = True               # bool(참) — 파이썬은 True/False 대문자
empty = None            # 값 없음(JS의 null)

# ── 4. f-string: 변수 끼워넣기 ──────────────────
name = "철수"
msg = f"안녕 {name}, {num}살"   # f"..." 안 {변수} = 값 삽입 → "안녕 철수, 42살"

# ── 5. 조건문: 콜론+들여쓰기 ────────────────────
if num >= 20:           # 조건 뒤 콜론(:), 다음 줄 4칸 들여쓰기 = 블록
    print("성인")
elif num >= 13:         # else if = elif
    print("청소년")
else:
    print("어린이")

# ── 6. 반복문 ───────────────────────────────────
for i in range(3):      # range(3) = 0,1,2 순서대로 i에 넣음
    print(i)
for fruit in ["사과", "배"]:  # 리스트 요소 하나씩 꺼내기
    print(fruit)
while count < 5:        # 조건 참인 동안 반복
    count += 1

# ── 7. 함수: def ────────────────────────────────
def add(a, b):          # def 이름(매개변수): 로 정의
    return a + b        # return = 결과 반환
def greet(n="손님"):     # n="손님" = 기본값. 안 넘기면 "손님"
    return f"{n}님"
add(2, 3)               # 호출 → 5

# ── 8. 자료구조 4종 ─────────────────────────────
nums = [1, 2, 3]        # 리스트(순서O, 수정O) — JS 배열
nums.append(4)          # 끝에 추가 → [1,2,3,4]
point = (10, 20)        # 튜플(순서O, 수정X) — 고정 묶음
user = {"name": "철수", "age": 20}  # 딕셔너리(키:값) — JS 객체
user["age"] = 21        # 딕셔너리 값 변경
unique = {1, 2, 2, 3}   # 집합(중복 제거) → {1,2,3}

# ── 9. 컴프리헨션: 한 줄 반복 ───────────────────
doubled = [n * 2 for n in nums]        # 각 요소 변형 → [2,4,6,8]
evens = [n for n in nums if n % 2 == 0]  # 조건 참인 것만 → [2,4]

# ── 10. 클래스: 데이터+기능 묶음 ────────────────
class Todo:                     # class 이름:
    def __init__(self, title):  # __init__ = 생성자(객체 만들 때 자동 실행)
        self.title = title      # self = 자기 자신. 객체에 값 저장
        self.done = False
    def toggle(self):           # 메서드(클래스 안 함수). 첫 인자 항상 self
        self.done = not self.done  # not = 참↔거짓 뒤집기

t = Todo("공부")                 # 객체 생성(→ __init__ 실행)
t.toggle()                      # t.done = True

# ── 11. 예외처리 ────────────────────────────────
try:                            # 실패할 수 있는 코드
    x = 10 / 0                  # 0으로 나눔 → 에러 발생
except ZeroDivisionError as e:  # 그 에러 종류를 잡아서 e에 담음
    print("0으로 못 나눔:", e)
꼭 알아야 할 지식: ①들여쓰기가 문법 — 4칸 스페이스로 블록 구분(안 맞으면 에러) ②변수에 타입 안 씀(동적) — 대신 실수 잦아 타입힌트 def f(x: int) 권장 ③True/False/None 대문자 ④리스트[]=수정O·튜플()=수정X·딕셔너리{}=키값 ⑤self는 메서드 첫 인자로 항상 명시 ⑥가상환경(venv)으로 프로젝트별 패키지 격리 필수.
쉽게: venv = 프로젝트마다 따로 쓰는 도구상자(버전 충돌 방지). 필수 습관.
