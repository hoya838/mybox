핵심 개념

1. 라우터 — URL과 함수 연결

from fastapi import FastAPI
app = FastAPI()

@app.get("/todos")           # GET /todos 요청 오면
def list_todos():
    return [{"title": "공부"}]   # 자동 JSON 변환

@app.get("/todos/{id}")      # 경로 파라미터
def get_one(id: int):
    return {"id": id}
쉽게: @app.get("/주소") = "이 주소로 오면 이 함수 실행". 데코레이터로 연결.

2. Pydantic — 입력 검증

from pydantic import BaseModel
class Todo(BaseModel):
    title: str
    done: bool = False

@app.post("/todos")
def add(t: Todo):     # 자동으로 형식 검사
    return t          # 잘못된 데이터는 자동 422 에러
쉽게: 들어오는 데이터 모양을 미리 정하면 자동 검사. 이상하면 알아서 막음.

3. 자동 문서 (강점)

코드만 쓰면 /docs에 API 문서가 자동 생성. 바로 테스트 가능.

쉽게: 서버 켜고 localhost:8000/docs 열면 메뉴판이 저절로. FastAPI 킬러 기능.

4. DB 연동 + 실행

pip install fastapi uvicorn sqlalchemy psycopg2
uvicorn main:app --reload   # 서버 실행
쉽게: SQLAlchemy로 PostgreSQL 연결, uvicorn으로 서버 켬. --reload=코드 바뀌면 자동 재시작.
