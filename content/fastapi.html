핵심 개념

1. 라우터 — URL과 함수 연결

from fastapi import FastAPI
app = FastAPI()

@app.get("/todos")           # GET /todos 요청 오면
def list_todos():
    return [{"title": "공부"}]   # 자동 JSON 변환

@app.get("/todos/{id}")      # 경로 파라미터
def get_one(id: int):
    return {"id": id}
쉽게: @app.get("/주소") = "이 주소로 오면 이 함수 실행". 데코레이터로 연결.

2. Pydantic — 입력 검증

from pydantic import BaseModel
class Todo(BaseModel):
    title: str
    done: bool = False

@app.post("/todos")
def add(t: Todo):     # 자동으로 형식 검사
    return t          # 잘못된 데이터는 자동 422 에러
쉽게: 들어오는 데이터 모양을 미리 정하면 자동 검사. 이상하면 알아서 막음.

3. 자동 문서 (강점)

코드만 쓰면 /docs에 API 문서가 자동 생성. 바로 테스트 가능.

쉽게: 서버 켜고 localhost:8000/docs 열면 메뉴판이 저절로. FastAPI 킬러 기능.

4. DB 연동 + 실행

pip install fastapi uvicorn sqlalchemy psycopg2
uvicorn main:app --reload   # 서버 실행
쉽게: SQLAlchemy로 PostgreSQL 연결, uvicorn으로 서버 켬. --reload=코드 바뀌면 자동 재시작.

필수 패턴 — 줄별 주석

FastAPI = 데코레이터로 경로 연결 + Pydantic으로 입출력 자동 검증 + /docs 문서 자동 생성. 주석 = 그 줄이 하는 일.

# ── 앱 만들기 ───────────────────────────────────
from fastapi import FastAPI          # 프레임워크 임포트
from pydantic import BaseModel       # 데이터 검증용 베이스

app = FastAPI()                      # 앱 객체 생성 — 모든 라우트가 여기 붙음

# ── Pydantic 모델: 입출력 계약 ──────────────────
class PostIn(BaseModel):             # 요청 본문 모양 정의
    title: str                       # title은 문자열 필수(없으면 자동 422 에러)
    content: str = ""                # 기본값 있으면 선택 항목

class PostOut(BaseModel):            # 응답 모양
    id: int
    title: str

# ── 라우트: 데코레이터로 URL+메서드 연결 ────────
@app.get("/posts")                   # GET /posts 요청을 아래 함수에 연결
async def list_posts():              # async = 비동기(DB 대기 중 다른 요청 처리)
    return [{"id": 1, "title": "hi"}] # 반환값이 자동으로 JSON 변환

@app.get("/posts/{post_id}")         # {post_id} = 경로 변수
async def get_post(post_id: int):    # 타입 int → 문자열 아니면 자동 422
    return {"id": post_id}

@app.post("/posts", response_model=PostOut, status_code=201)
#  response_model=응답 검증·문서화, status_code=201(생성됨)
async def create_post(post: PostIn): # 본문을 PostIn으로 자동 파싱·검증
    return {"id": 1, "title": post.title}  # PostOut 모양대로 반환

# ── 쿼리 파라미터·의존성 주입 ───────────────────
@app.get("/search")
async def search(q: str, limit: int = 10):  # ?q=..&limit=.. URL 쿼리 자동 매핑
    return {"q": q, "limit": limit}          # limit 기본 10

# 실행: uvicorn main:app --reload   → http://127.0.0.1:8000/docs 자동 문서
꼭 알아야 할 지식: ①Pydantic 모델이 검증·문서를 자동 — 타입 안 맞으면 손 안 대도 422 응답 ②경로 {id}=path, 함수 인자 기본값=query 파라미터 ③async def + 비동기 드라이버(asyncpg)로 동시성 ④response_model로 응답에서 민감필드(비번) 제외 ⑤/docs(Swagger)·/redoc 자동 생성 — 별도 문서화 불필요 ⑥DB 세션은 Depends() 의존성 주입으로 요청마다 관리.
